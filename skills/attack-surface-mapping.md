# Attack Surface Mapping

## Purpose
Ensure robust security practices in all deliverables.

## Principles
- Apply defense-in-depth strategies
- Follow OWASP guidelines and CWE top 25
- Validate all inputs, sanitize all outputs
- Use least-privilege access patterns
- Document security decisions and threat models

## Practices
- Perform threat modeling before implementation
- Run SAST/DAST scans on every change
- Review dependencies for known vulnerabilities
- Apply secure coding standards (no hardcoded secrets, parameterized queries)
- Include security test cases in every sprint
