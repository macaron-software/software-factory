# RTK Integration for OpenCode

This project uses **rtk** (Rust Token Killer) to reduce LLM token consumption by 60-90%.

## Quick Commands

Instead of running commands directly, prefix them with `rtk`:

| Standard | RTK | Savings |
|----------|-----|---------|
| `git status` | `rtk git status` | ~80% |
| `git diff` | `rtk git diff` | ~75% |
| `git log` | `rtk git log` | ~80% |
| `ls` | `rtk ls` | ~80% |
| `cat file` | `rtk read file` | ~70% |
| `grep pattern .` | `rtk grep pattern .` | ~80% |
| `cargo test` | `rtk cargo test` | ~90% |
| `pytest` | `rtk pytest` | ~90% |

## Supported Commands

- **Git**: status, diff, log, add, commit, push, pull, branch, fetch, stash
- **Files**: ls, read, find, grep
- **Tests**: cargo test, pytest, vitest, go test
- **Lint**: ruff, eslint, golangci-lint
- **Docker**: ps, images, logs
- **More**: See `rtk --help`

## Benefits

- **Token savings**: 60-90% reduction on command outputs
- **Same functionality**: Commands work identically, output is just optimized
- **No learning curve**: Just add `rtk` prefix

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/rtk-ai/rtk/refs/heads/master/install.sh | sh
```

Verify: `rtk --version` should show 0.24.0 or higher.
