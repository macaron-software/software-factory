SECURITY FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Encryption
  ✅ AES-256-GCM: Authenticated encryption
  ✅ Nonce: 12-byte random per message
  ✅ Format: [nonce(12)] || [ciphertext] || [tag(implicit)]
  ✅ No AAD: Plain authenticated encryption

Key Derivation
  ✅ Argon2id: Memory-hard KDF (resistant to GPU attacks)
  ✅ Output: 256-bit key + 16-byte salt
  ✅ Deterministic: Same passphrase + salt = same key
  ✅ Randomness: Different salt per derivation (unless provided)

Memory Safety
  ✅ zeroize: Securely clear key from memory (protection against cold boot)

Authentication
  ✅ JWT: HMAC-SHA256 signatures
  ✅ Claims: {sub: device_id, iat, exp}
  ✅ Expiry: 90 days
  ✅ Transport: Bearer token in Authorization header

Authorization
  ✅ RegisterDevice: Public, rate-limited (5/min per IP)
  ✅ Push: Requires JWT (60/min per device_id)
  ✅ Pull: Requires JWT (120/min per device_id)
  ✅ Health: Public (no auth)

Rate Limiting
  ✅ Per IP (registration abuse prevention)
  ✅ Per device_id (sync abuse prevention)
  ✅ Sliding window: Timestamps in memory, cleanup on check
  ✅ Configurable: (key, max_requests, window_secs)

CORS
  ✅ Allowed origins: baby.macaron-software.com, localhost:5173
  ✅ Methods: POST, OPTIONS
  ✅ Headers: Content-Type, Authorization, x-grpc-web, x-user-agent
  ✅ Exposed: grpc-status, grpc-message
  ✅ Max age: 3600 seconds
