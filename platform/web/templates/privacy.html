3. Lawful Basis (GDPR Art. 6)

4. Your Rights (GDPR Art. 15-20)

  • Access — Export your data: GET /api/me/export
  • Erasure — Delete your account: DELETE /api/me or Settings page
  • Portability — Export returns JSON format for portability
  • Rectification — Update profile in Settings
  • Objection — Contact privacy@macaron-software.com

5. Data Security (GDPR Art. 32)

TLS encryption in transit. bcrypt password hashing. Role-based access control (RBAC). Parameterized SQL queries. No secrets in source code.

6. Data Transfers

LLM API calls may transmit prompts to Azure OpenAI (EU region), MiniMax (China), or local MLX (no transfer). No user PII is sent to LLM providers.

7. Breach Notification (GDPR Art. 33)

In the event of a data breach, we will notify the supervisory authority within 72 hours and affected users without undue delay.
