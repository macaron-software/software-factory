# TDD Task - Catalog Route Implementation

## Anomaly
Tests navigate to `/catalog` route which doesn't exist in the frontend routes structure.

## Location
- Tests: `tests/journeys/booking.journey.spec.ts`, `tests/journeys/subscription.journey.spec.ts`
- Route: `src/routes/(app)/catalog/+page.svelte` (missing)

## Steps

### Phase 1: Create Catalog Route (RED)
```bash
# Create catalog route directory
mkdir -p src/routes/\(app\)/catalog
touch src/routes/\(app\)/catalog/+page.svelte
touch src/routes/\(app\)/catalog/+page.ts
```
