# TDD Task - Booking Stepper Component

## Anomaly
Tests verify booking stepper functionality with selectors (`step-station`, `step-bike`, `step-slot`, `step-confirm`, `booking-next`, `booking-back`) that need implementation.

## Location
- Tests: `tests/journeys/booking.journey.spec.ts`
- Components: `src/lib/components/booking/*.svelte`

## Steps

### Phase 1: Create Booking Stepper Component (RED)
```bash
# Create booking stepper component
touch src/lib/components/booking/BookingStepper.svelte
```

```svelte
<!-- src/lib/components/booking/BookingStepper.svelte -->
<script lang="ts">
  let { currentStep = 1, steps = ['Station', 'Vélo', 'Créneau', 'Confirmation'] }: Props = $props();
</script>

<div data-testid="booking-stepper" class="booking-stepper">
  {#each steps as step, i}
    <div
      data-testid="step-{step.toLowerCase()}"
      class="step"
      class:active={i + 1 === currentStep}
      class:completed={i + 1 < currentStep}
    >
      <span class="step-number">{i + 1}</span>
      <span class="step-label">{step}</span>
    </div>
  {/each}
</div>

<div class="booking-actions">
  <button data-testid="booking-back" class="btn-back" disabled={currentStep === 1}>
    Retour
  </button>
  <button data-testid="booking-next" class="btn-next" disabled={currentStep === steps.length}>
    Suivant
  </button>
</div>

<div data-testid="booking-summary" class="booking-summary">
  <div data-testid="summary-station" class="summary-station"></div>
  <div data-testid="summary-bike" class="summary-bike"></div>
  <div data-testid="summary-slot" class="summary-slot"></div>
</div>
```

### Phase 2: Verify Tests Pass

cd veligo-platform/frontend && npm run test:journey -- --grep "stepper\|booking"
